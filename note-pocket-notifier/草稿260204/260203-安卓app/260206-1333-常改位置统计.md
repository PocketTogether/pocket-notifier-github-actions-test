### app/build.gradle.kts
```
app/build.gradle.kts

修改应用ID
android.defaultConfig.applicationId
applicationId = "top.sakiko.pocket_notifier"

修改版本数字
android.defaultConfig.versionCode
versionCode = 1

修改版本号
android.defaultConfig.versionName
versionName = "0.0.1"
```

应用ID
```
- 必须包含至少一个点号 `.`
- 只能使用字母、数字、下划线 `_` 和点号 `.`
- 不需要是真实域名，也不需要你拥有这个域名
- 只要 applicationId 不同，系统就会把它当成完全独立的应用
```

版本号规范（versionCode / versionName）
```
- versionCode：必须是整数，每次发布新版本都要递增
  例如：1 → 2 → 3  
  Android 用它判断“哪个版本更 新”。

- versionName：给用户看的版本号，格式随意
  例如：`0.0.1`、`1.2.0`、`2025.01.01`  
  只要是字符串就行，没有强制格式。
```
